◆ move()

void s21::game::move ( )

Функция передвигает змейку по полю

Функция проверяет значение переменной action. И, в зависимости от результата, передвигает змейку по игровому полю. Если координаты первого элемента змейки совпадают с координатами яблока, происходит увеличение змейки и генерауция нового яблока
